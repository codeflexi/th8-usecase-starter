1 Preparation Checklist (ก่อนเข้าห้อง)

1. Know the "Villain" (ปัญหา)

  • เช็คข่าวบริษัท: ปีนี้เขาเน้น "ลดต้นทุน" หรือ "ขยายสาขา"?
  • ถ้าลดต้นทุน -> เน้นเรื่อง Fraud/Price Check
  • ถ้าขยายสาขา -> เน้นเรื่อง Speed/SLA

2. Setup Demo Data

  • เปลี่ยนชื่อ Vendor ใน Demo เป็นชื่อคู่ค้าจริงในวงการเขา
  • เปลี่ยนชื่อ Item เป็นวัตถุดิบหลักของเขา (เช่น เหล็ก, แป้ง, ชิป)
  • Tip: ใช้ไฟล์ Industry Demo Pack ที่เตรียมให้

2 The Hook (5 นาทีแรก - เจาะใจ CEO)

Target: CEO / MD Goal: ขาย "การควบคุม (Control)" ไม่ใช่ซอฟต์แวร์

"ท่านครับ... ในองค์กรระดับ 500 ล้านอย่างเรา ผมเชื่อว่าเรามี ERP ที่ดีอย่าง SAP/Oracle อยู่แล้ว

แต่คำถามคือ... ทำไมเรายังต้องมีคนมานั่งเซ็นเอกสารวันละเป็นร้อยใบ?
และทำไมทุกปียังมีเคส 'ซื้อของแพงเกินจริง' หรือ 'ของขาดสต็อก' หลุดรอดสายตาไปได้?

วันนี้ผมไม่ได้มาเสนอโปรแกรมทำเอกสารเพิ่มครับ
แต่ผมมาเสนอ 'Decision Control Center'... ระบบที่จะทำให้ Policy ที่ท่านเซ็นอนุมัติในห้องประชุม กลายเป็น AI Agent ที่เฝ้าประตูบริษัทให้ท่าน 24 ชั่วโมงครับ"

Action เปิดหน้า Dashboard โชว์ตัวเลข "Risk Saved" ตัวใหญ่ๆ ให้เห็นผลลัพธ์ก่อน

3 The Solution Story (เจาะลึก 3 แกน)

Target: CFO Goal: ขาย "ความโปร่งใส (Auditability)"
Screen เปิดหน้า End-to-End Flow (Step 2: AI Analysis)

"สำหรับมุมมองการเงิน สิ่งที่น่ากลัวที่สุดไม่ใช่การ 'จ่ายเงิน' แต่คือการ 'จ่ายโดยไม่รู้ว่าผิด'

(ชี้ที่หน้าจอ) ระบบ TH8 ใช้ AI + Rule Engine ครับ...
ตรงนี้ครับ (ชี้ Rule) กฎบอกว่า 'ห้ามเกิน 3 ล้าน' ระบบล็อกทันที ไม่มีหยวน นี่คือความแม่นยำ 100%
และตรงนี้ครับ (ชี้ Evidence/RAG) AI ไปดึงสัญญาเก่าปี 2023 มาเทียบให้เห็นเลยว่า ราคานี้ 'ปรับขึ้นได้' ตามสัญญาข้อ 4.2

CFO จะไม่ต้องเสียเวลาเปิดแฟ้มสัญญาเองอีกต่อไปครับ ข้อมูลมากองอยู่ตรงหน้าพร้อมตัดสินใจทันที"

Target: COO Goal: ขาย "ความเร็ว (Speed/SLA)"
Screen เปิดหน้า Ingest Simulator (SLA Alert)

"ในมุมปฏิบัติการ ผมทราบดีว่า 'ความล่าช้า = ต้นทุน' ครับ
ระบบนี้มี SLA Watchdog ที่จะเห่าเตือนทันทีถ้างานค้างนานเกินกำหนด

(กด Simulate Ingest) เห็นไหมครับ? พอ SLA เหลือต่ำกว่า 6 ชม. ระบบ Alert ไปที่มือถือ Manager ทันที
เราเปลี่ยนจากการ 'ตามแก้ปัญหาตอนของขาด' เป็นการ 'รู้ล่วงหน้าและป้องกันทัน' ครับ"

Target: IT Director Goal: ขาย "ความง่ายและปลอดภัย (Low Burden)"
Screen เปิดหน้า Audit Log (JSON View) หรือโชว์ Security Whitepaper

"ผมทราบว่าทาง IT งานล้นมืออยู่แล้วครับ... TH8 ถูกออกแบบมาเป็น Low-Touch Integration
เราคุยกันด้วย API JSON มาตรฐาน (ชี้ JSON) ไม่ต้องแก้ Core ERP เดิม

และที่สำคัญที่สุด... (หยิบ Security Paper ขึ้นมา) 'We DO NOT train AI on your data'
ข้อมูลของท่านถูกเก็บแยก (Single Tenant) และ Logic ทั้งหมดเป็น Config File (YAML) ที่ Business แก้เองได้ ไม่ต้องเปิด Ticket ให้ IT แก้โค้ดทุกครั้งที่มีโปรโมชั่นใหม่ครับ"

4 The Closing (สรุปจบให้ได้งาน)

Call to Action: "The Audit Challenge"

"ท่านครับ... การเปลี่ยนระบบทั้งบริษัทอาจดูเป็นเรื่องใหญ่
ดังนั้น วันนี้ผมขอเสนอ 'Proof of Value' ง่ายๆ ครับ

ขอให้ท่านส่งข้อมูลการจัดซื้อย้อนหลัง 3 เดือน (แบบปิดชื่อ) มาให้เรา
เราจะใช้ TH8 Audit ให้ฟรีๆ ภายใน 3 วัน...
ถ้าเราไม่เจอจุดรั่วไหล หรือจุดที่ท่านเสียเงินฟรีๆ... ท่านไม่ต้องซื้อครับ
แต่ถ้าเราเจอ... เราค่อยมาคุยเรื่อง Implementation Plan กัน

ท่านสะดวกให้ทีมผมเริ่มทำ NDA วันนี้เลยไหมครับ?"
