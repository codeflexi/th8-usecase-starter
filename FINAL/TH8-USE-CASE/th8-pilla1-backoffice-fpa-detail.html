Flow การทำงานของ FP&A Agent (สรุป)

  1. รวมข้อมูล GL, Sales, Budget, Forecast จากระบบแสงไทยและ SolarWing เข้าสู่ Data Warehouse
  2. คำนวณ P&L Consolidated + Variance vs Budget / Last year
  3. อ่าน config จาก scenario_assumption เพื่อจำลอง Scenario ต่าง ๆ
  4. สร้างรายงาน & Dashboard สำหรับผู้บริหาร + ตอบคำถามผ่าน FP&A Agent
  5. เก็บ Log การใช้งานและผลลัพธ์ เพื่อปรับ model / assumption รอบถัดไป
